# Smart Panchayat Assistant

Smart Panchayat Assistant is an NLP-based system that helps citizens understand government schemes in simple language.

## Features
- Ask questions about government schemes
- Supports multiple languages
- Uses Retrieval-Augmented Generation (RAG)

## Technologies
- Python
- LangChain
- FAISS
- Sentence Transformers
- FLAN-T5

## Example

Input:
What is PM Kisan scheme?

Output:
PM Kisan scheme provides ₹6000 per year as financial assistance to farmers.

## Run the project 
Install dependencies :
pip install -r requirements.txt

Run the assistant:
python panchayat_ai.py
